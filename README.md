# AI GitHub Repository Analyzer

## Description

AI GitHub Repository Analyzer is an intelligent developer tool that analyzes any GitHub repository, identifies the technology stack, project architecture, code complexity, repository health, and documentation quality, then generates actionable insights, improvement suggestions, and concise project reports to help developers, recruiters, and learners understand codebases faster.

Developers, students, and recruiters often spend hours exploring unfamiliar GitHub repositories to understand their purpose, architecture, technology stack, code quality, and documentation. This tool automates repository analysis and generates clear, AI-powered insights, summaries, and recommendations, reducing onboarding and evaluation time from hours to minutes.

## Tech Stack

Next.js, React, TypeScript, Tailwind CSS, Node.js, Express.js, GitHub API, OpenAI API, PostgreSQL, Prisma ORM, Redis, Docker

## Features

- Analyze public GitHub repositories from a URL
- Detect programming languages and technology stack
- Generate AI-powered repository summaries
- Visualize project architecture and folder structure
- Evaluate code complexity and maintainability
- Analyze README quality and documentation coverage
- Identify code smells and improvement opportunities
- Generate onboarding guides for new contributors
- Provide repository health and quality scores
- Export analysis reports as PDF or Markdown

## Installation

```bash
Clone the repository
Navigate to the project directory
Install dependencies using npm install
Create a .env file and add required API keys
Run database migrations
Start the development server with npm run dev
Open the application in your browser
Enter a GitHub repository URL to begin analysis
```
